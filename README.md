# 🍌 Nano Banana - Air Quality Monitoring

Proyek ini adalah *Dashboard Real-time* untuk memantau kualitas udara (Indoor & Outdoor) menggunakan sensor MQ-135, MQ-7, PM 2.5, dan PM 10 berbasis ESP32.

### 🛠️ Teknologi yang Digunakan:
* **Hardware:** ESP32, Sensor MQ Series, Sensor PMS5003
* **Database:** Firebase Realtime Database
* **Frontend:** HTML, CSS (Glassmorphism), JavaScript
* **Visualisasi Data:** Chart.js (Grafik) & ECharts (Speedometer ISPU)

### 👨‍⚕️ Pengembang
Dikembangkan untuk kebutuhan penelitian ilmiah (KTI) / Pemantauan Kesehatan Lingkungan.
